How each defense works
Adversarial training
Augments training data with adversarial examples so the model learns to handle them. Highest accuracy recovery, computationally intensive.
Autoencoder denoising
Passes input through an autoencoder that strips adversarial noise before the classifier sees it. No model retraining needed.
Input preprocessing
Feature squeezing, Gaussian smoothing, and JPEG compression reduce perturbation magnitude before inference. Fast to deploy.
Defensive distillation
Trains a student model on soft probability outputs from a teacher model. Gradient masking makes gradient-based attacks less effective.
